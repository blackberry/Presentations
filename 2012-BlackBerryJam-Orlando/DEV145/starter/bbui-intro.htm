bbUI.js Introduction

bbUI.js is an open source (https://github.com/blackberry/bbUI.js) JavaScript toolkit designed to enable HTML5 + BlackBerry WebWorks applications to have a look and feel similar to native BlackBerry applications with minimal effort.

bbUI.js provides common UI constructs that are found on the BlackBerry operating system, allowing developers to create HTML5 + WebWorks applications that follows the UI guidelines of the target OS.

  2. Verify how the UI / layout of this application changes to accomodate the Tablet OS UI guidelines.
