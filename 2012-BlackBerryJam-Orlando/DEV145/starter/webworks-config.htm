Beginner: Define application properties
  1. Edit config.xml
  2. Modify <name> and <author> elements.
  3. Verify changes in Ripple using Information panel.
  4. Test changes on simulator or live device, verifying home screen application name is correct.
Intermediate: Create home screen icon
  1. Create an 86x86 PNG home screen icon named icon.png.
  2. See http://www.orison.biz/apps/playbook-icon-maker/ for help.
  3. Save icon to this local webworks project.
  4. Ensure src property of <icon> element in config.xml is correct.
  5. Verify changes in Ripple using Information panel.
  6. Test changes on simulator or live device, verifying home screen icon is correct.
Advanced: White listing external domains
  1. Edit config.xml
  2. Replace the "index.htm" value of the <content> element src property with "http://devblog.blackberry.com".
  3. Add an <access> element to config.xml to whitelist the "http://devblog.blackberry.com" domain correctly. See http://developer.blackberry.com/html5/documentation for help ("Developing" tutorial).
  4. Test changes on simulator or live device. Verify whether any content from external domains has not been loaded.
  5. Add more <access> elements to config.xml to whitelist any remaining external domains used by "http://devblog.blackberry.com" (e.g. http://www.twitter.com).
  6. Test changes on simulator or live device. Verify all external content has been loaded correctly.
