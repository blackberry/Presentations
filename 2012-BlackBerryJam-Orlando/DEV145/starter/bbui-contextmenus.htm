Context Menus (BlackBerry 10 only)

BlackBerry 10 allows for a press and hold context menu that is very similar to the action bar overflow menu.

Peeking the context menu will show the row of action icons that can be clicked and part of the context information in the header of the menu. When the user swipes from right to left it will pull the full menu into view if they want to see the text labels for all the items.

Display the full context menu
Tasks:
  1. Test using Ripple, select "BlackBerry 10 WebWorks" platform and click on "Show Menu" image button
  2. Edit bbui-contextmenu.htm
  3. Add 3 menu items to the <div id="contextMenu" data-bb-type="context-menu"> element
  4. Test using Ripple, verify newly added items appear when the context menu is opened
  5. Add a second image item to the <div data-bb-type="image-list"> element having data-bb-title="Peek Menu"
  6. Assign onclick="peekContextMenu()" to this image item
  7. Test using Ripple, verify only context menu icons are displayed when selecting the "Peek Menu" image item
  8. Edit js/bbui-contextmenu.js
  9. Create a clickHandler(param) method to handle context menu item selections.
  10. Display a message to the user indicating which context menu item was selected.
  11. Assign onclick="clickHandler('ABC')" to each of the context menu items within the <div id="contextMenu" data-bb-type="context-menu"> element
  12. Test using Ripple, verify the clickHandler message is displayed when selecting items from the context menu

Tips:

  • See https://github.com/blackberry/bbUI.js for documentation
  • bb.popScreen() will always remove the last screen added to the display stack.
