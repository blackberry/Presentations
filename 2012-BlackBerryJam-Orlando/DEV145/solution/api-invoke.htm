blackberry.invoke Examples

This example will demonstrate how to launch external applications using the blackberry.invoke WebWorks API.

  1. Add <feature> elements for blackberry.invoke and blackberry.invoke.BrowserArguments to config.xml
  2. Read URL from input field below
  3. Launch browser to target url when user clicks 'Open' button
