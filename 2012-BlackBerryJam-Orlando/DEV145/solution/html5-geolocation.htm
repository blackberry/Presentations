GPS Examples

This example will demonstrate how to retrieve the GPS coordinates for the user.

  1. Call navigator.geolocation.getPosition()
  2. Create onSuccess(position) and onFail(error) callback methods to handle responses
  3. Display updated information to the screen when user clicks 'Refresh' button.
