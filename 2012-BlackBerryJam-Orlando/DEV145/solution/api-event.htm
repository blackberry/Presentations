blackberry.event Examples

This example will demonstrate how to handle the application pause/resume events using the blackberry.event WebWorks API.

  1. Add a <feature> element for blackberry.event to config.xml
  2. Use blackberry.event.addEventListener() to create handlers for the pause and resume events
  3. Create doPause() and doResume() methods that display a message for the given event

Tips:

  • Use Ripple events panel to maually fire the pause and resume
  • Using a live device, move the application to the background (without closing) to initiate the pause event
  • See http://developer.blackberry.com/html5/api
