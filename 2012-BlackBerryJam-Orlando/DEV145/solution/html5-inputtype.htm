Input Type Examples

This example demonstrates how HTML5 input types can be used to target different virtual keyboard variants:

  1. Add more input fields to this page, similar the following row.
  2. Change the value of the type property from 'text' to any of the following:
    • tel
    • url
    • email
    • number
    • date
    • time
    • datetime
    • color
  3. Using simulator or live device, test which virtual keyboard is displayed.
